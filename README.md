# BL_Jwt_Server_Net8

A .NET 8 server project demonstrating how to generate and validate JWT tokens.

## Features
- JWT token generation
- Token validation middleware
- Secure endpoints
- Clean and minimal implementation

## Technologies
- .NET 8
- C#
- JWT Authentication

## Running the Project
Update the JWT secret in `appsettings.json` and run:

    dontet run
